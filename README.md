# Hospital Resource Management System (MediCore)

**Live Production Deployment:** [https://hospital-resource-manager-nfsrvl2p3-poorvajan-g-s-projects.vercel.app](https://hospital-resource-manager-nfsrvl2p3-poorvajan-g-s-projects.vercel.app)

## Overview
A comprehensive Next.js web application for hospital resource management, including:
- Patient Registrations and Scheduling
- Staff and Department Management
- Real-time Inventory and Blood Bank Monitoring
- Medical Records

*Built with Next.js, Tailwind CSS, TypeScript, and a Neon PostgreSQL database.*
